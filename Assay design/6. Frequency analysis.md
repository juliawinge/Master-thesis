# Frequency analysis
*This step describes how to check the tissue distribution of mutations within selected target regions.* <br>

*The analysis was performed only for sites selected for primer design. The aim was to evaluate whether mutations in the selected target site or region were more or less common in the intended target region.*

## 1. Download COSMIC files
For each gene of interest, download TSV files from COSMIC as previously described in section 1, selecting the dataset for all tissue types beneath the **Tissue distribution** table.   <br>
<img width="724" height="196" alt="image" src="https://github.com/user-attachments/assets/a1efb041-9979-4a26-8f00-d00a81243db6" />

Download both:
**Positive data**  <br> <img width="1289" height="365" alt="image" src="https://github.com/user-attachments/assets/b7eabecd-70f4-4722-b232-1714eefc5dcc" /> <br>
and **All data**   <br><img width="1423" height="411" alt="image" src="https://github.com/user-attachments/assets/92f5c216-3bb6-4eb4-aba8-c0e2a8aa307b" />

## 2. Filter by CDS mutations position
Open the downloaded TSV file in Excel as mentioned in section 1.  <br>

Start by analysing **All data**.  <br>

Click on the column header for CDS mutation and filter based on the CDS position of interest.  <br>

For target regions, enter all CDS positions included in the selected region.  <br>

Click **OK**. <br>
<img width="452" height="353" alt="image" src="https://github.com/user-attachments/assets/f7ceba21-0348-4029-afd3-8ddcedaaad81" />

## 3. Check tissue types with mutations in the target region
Press on the column header for **Tissue Type**.  <br>

This shows which tissue types have mutations in the selected target site or region.  <br>

Select one type and press **OK**. <br>
<img width="493" height="737" alt="image" src="https://github.com/user-attachments/assets/0a0c23fe-2622-415d-94bb-c49157ceccb0" />

## 4. Check the number of samples in each tissue type
After selecting a tissue type, the number of samples matching the filter will be noted in the bottom left corner.  <br>

Denote the number of samples within the target region for that tissue type.  <br>

Repeat this for all tissue types. If fewer than three mutations are found for a tissue type, skip that tissue and continue with the next one.  <br>
<img width="1040" height="582" alt="image" src="https://github.com/user-attachments/assets/6fd1cbcf-95e0-47dd-bc7b-4c6f126819a4" />

## 5. Check the total number of samples for each tissue type
Remove the CDS mutation filter.  <br>

Keep the tissue type filter and denote the total number of samples available for each tissue type.  <br>

This gives the total number of samples within each tissue type and is used to calculate the mutation frequency. <br> 
<img width="1112" height="147" alt="image" src="https://github.com/user-attachments/assets/d9bb7236-ccf3-405a-89f2-9aa16f3382ca" />

## 6. Repeat using positive mutation data
Add tissue type filter and denote the total number of samples available for each tissue type.  <br>

This gives the total number of mutated samples within each tissue type and is used to calculate the mutation frequency.  <br>
