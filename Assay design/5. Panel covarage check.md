# Check internal panels for target site overlap
*Validated hotspots and hot regions were checked against existing internal panels to avoid selecting target sites that had already been included in previous assay design.* <br>

If a site was already included in an in-house panel and performs well, no new assay design will be performed. <br>

If a site was already included in an in-house panel and showed lower performance, new assays could be designed to try to improve performance. <br>

If a site had already been designed and tested multiple times with low performance, no further assay design was performed.  <br>
