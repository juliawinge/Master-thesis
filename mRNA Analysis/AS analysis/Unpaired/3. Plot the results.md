# Create a Venn diagram of genes with differential alternative splicing using UBUNTU
*This step creates a Venn diagram showing the overlap of genes with significant alternative splicing (AS) events between ectopic endometrium, ovarian lesions, and peritoneal lesions compared to healthy controls.* <br>

*The analysis uses gene lists generated from filtered rMATS results. Each input file contains unique genes with at least one significant AS event (FDR < 0.005 and |ΔPSI| ≥ 0.2).*

## 1. Change to the correct directory
Go to the directory containing the sorted gene list:
```
cd /your_directory/sample_download/your_dataset/rMATS/results
```
 <br>

 The directory should contain the map `sorted_vs`, which should contain: 
```
cntrl_eut.txt
cntrl_ect.txt
cntrl_ectovary.txt
```

## 2. Create the plotting script
Create a Python script to visualize the overlap of genes with significant AS events:
```
nano create_venn_disease_ vs_control.py
```
 <br>
 
Paste the following code:
```
import matplotlib.pyplot as plt
from matplotlib_venn import venn3, venn2
import matplotlib.font_manager as fm

# ===== FONT =====
# Add Georgia font to match the thesis style.
georgia_path = "/usr/share/fonts/truetype/msttcorefonts/Georgia.ttf"
fm.fontManager.addfont(georgia_path)

plt.rcParams.update({
    "font.family": "Georgia",
    "font.size": 12,
    "axes.titlesize": 16,
    "pdf.fonttype": 42,
    "ps.fonttype": 42
})

# ===== FUNCTION =====
# Read one gene per line and store the genes as a set.
def read_genes(file):
    with open(file) as f:
        return set(line.strip() for line in f if line.strip())

# ====== DATA ======
# Load gene list from the different tissue comparisons.  
cntrl_eut = read_genes("sorted_vs/cntrl_eut.txt")
cntrl_ect = read_genes("sorted_vs/cntrl_ect.txt")
cntrl_ectovary = read_genes("sorted_vs/cntrl_ectovary.txt")

# ====== COLORS ======
venn_colors = ("#DD8452", "#4C72B0", "#55A868")

# ====== FIGURE ======
# Create a Venn diagram showing overlap between tissue types. 
plt.figure(figsize=(9,6))

v = venn3(
    [cntrl_eut, cntrl_ect, cntrl_ectovary],
    set_labels=(
        "Eutopic endometrium",
        "Peritoneal lesion",
        "Ovarian lesion"
    ),
    set_colors=venn_colors,
    alpha=0.45
)
plt.tight_layout()

# ===== SAVE =====
# Save figure as a PNG.
plt.savefig("venn_disease_vs_control.png", dpi=300)
plt.close()

print("Venn diagrams created!")
```
<br>

Save the file by pressing `Ctrl+O`, then `Enter`, followed by `Ctrl+X`.

## 4. Run the plotting script
```
python3 create_venn_disease_ vs_control.py
```

## 5. Output
The results will be saved in the analysis directory as:
`venn_disease_vs_control.png`

The figure visualizes the overlap of genes with significant alternative splicing events between ectopic endometrium, ovarian, and peritoneal lesions relative to healthy controls. Unique and shared genes are displayed in the corresponding Venn diagram regions.
