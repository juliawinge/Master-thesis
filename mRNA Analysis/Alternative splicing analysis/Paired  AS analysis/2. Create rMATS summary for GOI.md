# Create GOI summary from paired rMATS results using UBUNTU
*This step creates a summary table of raw and filtered alternative splicing (AS) events in selected genes of interest (GOIs) from patient-matched rMATS comparisons.* <br>

*The input folders are rMATS results from paired comparisons between ectopic lesion samples and the corresponding eutopic endometrium samples from the same patient.*

## 1. Go to the analysis directory
Go to the directory containing the patient-matched rMATS result:
```
cd /your_directory/sample_download/your_dataset/rMATS/results/paired_control
```
<br>

The directory should contain one folder for each paired comparison, for example:
```
P7_Ovary
P7_Ect
P13_Ovary
P13_Ect
P16_Ovary
P16_Ect
```
## 2. Create a gene of interest list
Create a text file containing the GOIs:
```
nano goi.txt
```
<br>
 
Add one gene per line:
```
SLC1A1
STAT3
TCHH
CYP4F11
ENOSF1
```
Save the file by pressing `Ctrl+O`, then `Enter`, followed by `Ctrl+X`.

## 3. Create the summary script
Create a shell script that summarizes raw and filtered rMATS events for each GOI:
```
nano create_goi_rmats_summary.sh
```
 <br>
 
Paste the following code:
```
#!/bin/bash

# File containing the genes of interest.
GOI="goi.txt"

# rMATS comparison folders included in the analysis.
# Each folder should contain rMATS output from one paired comparison.
COMPARISONS=(
  "P7_Ovary"
  "P13_Ovary"
  "P16_Ovary"
  "P17_Ovary"
  "P13_Ect"
  "P16_Ect"
  "P17_Ect"
)

# rMATS even types included in the analysis.
EVENTS=("SE" "A3SS" "A5SS" "MXE" "RI")

# Create output file and add column names.
echo -e "Gene\tComparison\tEvent_type\tRaw_events\tFiltered_events" > goi_rmats_summary.tsv

# Count raw and filtered events for each GOI, comparison, and event type. 
while read gene; do
  for comp in "${COMPARISONS[@]}"; do
    for ev in "${EVENTS[@]}"; do

      # Define input files for raw and filtered rMATS results. 
      raw_file="${comp}/${ev}.MATS.JCEC.txt"
      filt_file="${comp}/${ev}.MATS.JCEC.filtered.txt"

      # Set default count to zero. 
      raw_count=0
      filt_count=0

      # Count raw rMATS events where the gene name matches the GOI.
      if [ -f "$raw_file" ]; then
        raw_count=$(awk -v g="$gene" 'BEGIN{FS="\t"} NR>1 {gsub(/"/, "", $3); if ($3==g) count++} END{print count+0}' "$raw_file")
      fi

      # Count filtered rMATS events where the gene name matches the GOI.
      if [ -f "$filt_file" ]; then
        filt_count=$(awk -v g="$gene" 'BEGIN{FS="\t"} NR>1  {gsub(/"/, "", $3); if ($3==g) count++} END{print count+0}' "$filt_file")
      fi

      # Write counts in the summary file.
      echo -e "${gene}\t${comp}\t${ev}\t${raw_count}\t${filt_count}" >> goi_rmats_summary.tsv

    done
  done
done < "$GOI"

echo "Done. Output written to goi_rmats_summary.tsv"
```
<br>

Save the file by pressing `Ctrl+O`, then `Enter`, followed by `Ctrl+X`.

## 4. Make the script runnable
```
chmod +x create_goi_rmats_summary.sh
```

## 5. Run the summary script
```
./create_goi_rmats_summary.sh
```

## 6. Output
The results will be saved in the analysis directory as:
`goi_rmats_summary.tsv`

The output file lists the number of raw and filtered AS events for each GOI, comparison, and event type.
