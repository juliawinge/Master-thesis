# Selection of public RNA-seq datasets
*This step describes how publicly available endometriosis RNA-seq datasets were identified and selected from the NCBI Gene Expression Omnibus (GEO) database: https://www.ncbi.nlm.nih.gov/geo/*

## 1. Search GEO Datasets
Go to the GEO homepage: https://www.ncbi.nlm.nih.gov/geo/ 

Search using the keyword:`Endometriois` <br>
Apply the following filters
* Entry type: `Series`
* Organism: `Homo sapiens`
* Study type: `Expression profiling by high-throughput sequencing`
* Attribute name: `Tissue`

## 2. Data selection
Each dataset was manually evaluated according to the following inclusion criteria:
- Public datasets
- Endometriosis tissue 
- Endometriosis and control samples
- Bulk RNA sequensing
- No experimental treatment
- At least three samples per group
- Raw sequencing data available

## 3. Dataset evaluation
The following information was collected for each dataset  was denoted for each dataset:
- GEO accosion
- Tissue type
- Endometriois subtype
- Number of patients
- Number of controls
- Sequensing layout
- Libary source
- Sequensing platform
- Read length
- Relese date

Datasets were ranked according to overall sutibility for downstream analysis RNA-seq ananlysis.
