# Run grouped rMATS analysis using UBUNTU
*This step describes how to run rMATS for grouped comparisons between healthy control samples, ectopic endometrium, ovarian, and peritonial lesion samples.* <br>

*The analysis was performed to identify differentialy AS spliced gene between tissue groups. The filtered rMATS results were later used to generate gene lists for overlap analysis and Venn diagram visualization.*

## Requirements
rMATS must be installed and available. <br>
Aligned BAM files generated from the nf-core/rnaseq pipeline are required as input. <br>
A reference annotation file in `.gtf` format is required.

## 1. Go to the analysis directory
Go to the rMATS directory for the dataset:
```
cd /your_directory/sample_download/your_dataset/rMATS
```
<br>

## 2. Create group files
Create one `.csv` file for each sample group. Each file should contain the full path to the BAM files included in that group. <br>

For example:
```
nano group_cntrl.csv
```
Add one BAM file path per line:
```
/your_directory/sample_download/your_dataset/bam/Cntrl_P1.bam,
/your_directory/sample_download/your_dataset/bam/Cntrl_P4.bam,
/your_directory/sample_download/your_dataset/bam/Cntrl_P5.bam,
/your_directory/sample_download/your_dataset/bam/Cntrl_P6.bam,
/your_directory/sample_download/your_dataset/bam/Cntrl_P7.bam
```

Create group files for all of the following groups:
```
group_cntrl.csv
group_eut.csv
group_ect.csv
group_ectovary.csv
```
Save each file by pressing ``Ctrl+O`, then `Enter`, followed by `Ctrl+X`.

## 3. Run rMATS for grouped comparisons
Run rMATS using two group files as input. <br>

For example, to compare healthy controls with eutopic endometrium:
```
rmats.py \
--b1 /your_directory/sample_download/your_dataset/rMATS/group/group_cntrl.csv \
--b2 /your_directory/sample_download/your_dataset/rMATS/group/group_eut.csv \
--gtf /your_directory/sample_download/your_dataset/rMATS/reference/gencode.v49.primary_assembly.annotation.gtf \
-t single \
--readLength 100 \
--variable-read-length \
--nthread 4 \
--od /your_directory/sample_download/your_dataset/rMATS/results/results_cntrl_vs_eut \
--tmp /your_directory/sample_download/your_dataset/rMATS/results/results_cntrl_vs_eut/tmp
```
The output directory contains the rMATS results for the selected comparison. 

## 4. Repeat for all grouped comparisons
Repeat the analysis for all grouped comparisons in the overview analysis:

```
Control vs Eutopic endometrium -> results_cntrl_vs_eut
Control vs Peritonial lesion -> results_cntrl_vs_ect
Control vs Ovarian lesion -> results_cntrl_vs_ectovary
```

## 5. Filter significant AS events
Filtered files were generated using the following criteria. 
* FDR<0.05
* |ΔPSI| ≥ 0.2

For example, filter one grouped comparison by running: 
```
for ev in SE A3SS A5SS MXE RI; do
  awk 'BEGIN{FS=OFS="\t"} NR==1 || ($20 < 0.05 && ($23 >= 0.2 || $23 <= -0.2))' \
/your_directory/sample_download/your_dataset/rMATS/results/results_cntrl_vs_eut/${ev}.MATS.JCEC.txt \
  > /your_directory/sample_download/your_dataset/rMATS/results/results_cntrl_vs_eut/${ev}.MATS.JCEC.filtered.txt
done
```
## 6. Output
Each grouped comparison folder contains filtered rMATS result files for all event types: 

``` text
SE.MATS.JCEC.txt
SE.MATS.JCEC.filtered.txt

A3SS.MATS.JCEC.txt
A3SS.MATS.JCEC.filtered.txt

A5SS.MATS.JCEC.txt
A5SS.MATS.JCEC.filtered.txt

MXE.MATS.JCEC.txt
MXE.MATS.JCEC.filtered.txt

RI.MATS.JCEC.txt
RI.MATS.JCEC.filtered.txt
``` 
These files can be used for downstream analysis and visualisation.
