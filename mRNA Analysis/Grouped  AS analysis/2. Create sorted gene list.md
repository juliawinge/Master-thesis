# Create gene lists from grouped rMATS results using UBUNTU
*This step extracts genes with significant AS events from filtered rMATS results and creates sorted gene lists used for Venn diagram visualization.* <br>

*Each gene list contains unique genes with at least one significant AS event in the corresponding comparison.* 

## 1. Go to the analysis directory
Go to the rMATS directory containing the grouped rMATS results:
```
cd /your_directory/sample_download/your_dataset/rMATS/results
```
<br>

The directory could contain grouped comparison folders, for example:
```
results_cntrl_vs_eut
results_cntrl_vs_ect
results_cntrl_vs_ectovary
```
## 2. Extract genes from each AS event type
Go to one grouped comparison folder:
```
cd /your_directory/sample_download/your_dataset/rMATS/results/results_cntrl_vs_eut
```
Extract unique genes from each filtered rMATS event file: 
```
for ev in SE A3SS A5SS MXE RI; do
  cut -f3 ${ev}.MATS.JCEC.filtered.txt | \
  tail -n +2 | sort | uniq > ${ev}_genes.txt
  
  echo "$ev genes:"
  wc -l ${ev}_genes.txt
done
```

## 3. Combine genes across AS event types
Combine the gene list from all event types:
```
cat SE_genes.txt A3SS_genes.txt A5SS_genes.txt MXE_genes.txt RI_genes.txt | \
sort | uniq > ALL_genes_cntrl_vs_eut.txt

wc -l ALL_genes_cntrl_vs_ect.txt
```
<br>
This creates one gene list containing all unique genes with at least one filtered AS event in the comparison. 

## 4. Repeat for all grouped comparisons
Repeat the gene extraction and merging step for each group comparison folder: 
```
results_cntrl_vs_eut -> ALL_genes_cntrl_vs_eut.txt
results_cntrl_vs_ect -> ALL_genes_cntrl_vs_ect.txt
results_cntrl_vs_ectovary -> ALL_genes_cntrl_vs_ectovary.txt
```
## 5. Sort gene list for downstream comparisons
Go back to the results directory:
```
cd /your_directory/sample_download/your_dataset/rMATS/results
```
<br>

Sort each combined gene list and save it in a folder called `sorted_vs`:
```
sort results_cntrl_vs_eut/ALL_genes_cntrl_vs_eut.txt -o sorted_vs/cntrl_eut.txt
sort results_cntrl_vs_ect/ALL_genes_cntrl_vs_ect.txt -o sorted_vs/cntrl_ect.txt
sort results_cntrl_vs_ectovary/ALL_genes_cntrl_vs_ectovary.txt -o sorted_vs/cntrl_ectovary.txt
```

## 6. Output
The stored gene lists will be saved in the folder `sorted_vs` and will be used to create the Venn diagram.
