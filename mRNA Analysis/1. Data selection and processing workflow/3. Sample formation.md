# Convert SRR files to FASTQ using UBUNTU
*This step converts `.sra` files into compressed paired-end `.fastq.gz` files, which is the input type used in downstream RNA-seqanalysis. This step was performed on the server after transferring the downloaded `.sra` files from a laptop via an external drive or USB device.* 

## Requirements 
SRA Toolkit must be installed and available in the terminal:
fasterq-dump --version

## 1. Go to the dataset directory
Move to the directory dedicated to the downloaded `.sra` files:
```
cd /your_directory/sample_download/your_dataset/sra
```
## 2. Create a sample sheet
Create a tab-separated samplesheet:
```
nano samplesheet_srr_to_fastq.tx
```
<br>

Add the SRA_IDs and sample names. The file must be tab-separated:
```
SRA_ID SAMPLE_NAME
SRRxxxxxxxxx	xxx_Control
SRRxxxxxxxxx	xxx_OE
SRRxxxxxxxxx	xxx_OE
SRRxxxxxxxxx	xxx_OE
SRRxxxxxxxxx	xxx_OE
............ ......
```
<br>

Save the file with ```Ctrl + O```, press ```Enter```, and exit with ```Ctrl+X```.

## 3. Create a formatting script
Create a script for converting all samples (`SRR`-> `fastq.gz`) listed in the sample sheet:
```
nano srr_to_fastq_gz.sh
```
<br>

Paste the following code:
```
#!/bin/bash
set -euo pipefail

# Project directory
PROJECT="cd /your_directory/sample_download/your_dataset/sra"

# Sample sheet containing SRA IDs and sample names
SAMPLESHEET="$PROJECT/samplesheet_srr_to_fastq.txt"

# Create output and temporary directories
mkdir -p "$PROJECT/fastq" "$PROJECT/tmp"

# Skip header and read SRA IDs and sample names
tail -n +2 "$SAMPLESHEET" | while IFS=$'\t' read -r SRA_ID SAMPLE_NAME
do
    echo "================================"
    echo "Converting $SRA_ID -> $SAMPLE_NAME"
    echo "================================"

    # Convert SRA to paired-end FASTQ files
    fasterq-dump "$PROJECT/sra/$SRA_ID" \
        -O "$PROJECT/fastq" \
        --split-files \
        --temp "$PROJECT/tmp"

    # Rename files using sample names
    mv "$PROJECT/fastq/${SRA_ID}_1.fastq" \
       "$PROJECT/fastq/${SAMPLE_NAME}_R1.fastq"

    mv "$PROJECT/fastq/${SRA_ID}_2.fastq" \
       "$PROJECT/fastq/${SAMPLE_NAME}_R2.fastq"

    # Compress FASTQ files
    gzip -f "$PROJECT/fastq/${SAMPLE_NAME}_R1.fastq"
    gzip -f "$PROJECT/fastq/${SAMPLE_NAME}_R2.fastq"

    echo "Finished: $SAMPLE_NAME"
done

# Remove temporary files
rm -rf "$PROJECT/tmp"

echo "All SRR files have been converted."
```
<br>

Save the file with ```Ctrl + O```, press ```Enter```, and exit with ```Ctrl+X```.

## 4. Make the script runnable
```
chmod +x srr_to_fastq_gz.sh
```
## 5. Run the conversion
```
./srr_to_fastq_gz.sh
```
<br>

Converted paired-end `fastq.gz` files will be saved in :
`/your_directory/sample_download/your_dataset/sra`
