# STR analysis using ExpansionHunter Denovo (EHdn)
*This step applies ExpansionHunter Denovo (EDH) to aligned RNA-seq BAM files to investigate candidate short tandem repeat (STR) regions. Due to the limitations of RNA-seq data for STR detection, additional preprocessing of the aligned BAM files using GATK SplitNCigarReads was also evaluated.*

## Requirements
ExpansionHunter Denovo should be installed according to the official documentation: <br>
https://github.com/Illumina/ExpansionHunterDenovo/blob/master/documentation/02_Installation.md

Additional software: 
- GATK
- samtools

Input files:
- Reference genome in FASTA format. 
- Aligned BAM files


## 1. Go to the analysis directory
Go to the EHdn directory for the dataset:
```
cd /your_directory/sample_download/your_dataset/EHdn
```
<br>

## 2. Generate STR profiles
Generate an STR profile for each sample:
```
ExpansionHunterDenovo profile \
 --reads sample.bam \
 --reference reference_genome.fa \
 --output-prefix output/sample_name \
 --min-anchor-mapq 50 \
 --max-irr-mapq 40
```
Repeat this step for all samples. <br>

## 3. Create a manifest file
Create a manifest describing all case and control samples according to https://github.com/Illumina/ExpansionHunterDenovo/blob/master/documentation/06_Merging_profiles.md

Create one manifest file for each comparison. Control samples consisted of either healthy endometrium or patient-matched eutopic endometrium. 
```
Cntrl vs Eut
Cntrl vs Ect
Cntrl vs Ectovary
Eut vs Ect
Eut vs Ectovary
```

## 4. EHdn analysis
Run the downstream EHdn workflow according to the official documentation:
https://github.com/Illumina/ExpansionHunterDenovo/tree/master

Repeat for all manifests. 

## 5. Optional preprocessing
To improve compatibility with EHdn, STAR-aligned BAM files were additionally processed using GATK SplitNCigarReads before repeated analysis. 

```

gatk SplitNCigarReads \
  -R /your_directory/reference_genome.fa\
  -I /your_directory/sample_download/your_dataset/results/star_salmon/Endo_P7_EctOvary_bulk.markdup.sorted.bam\
  -O /your_directory/sample_download/your_dataset/results/splitted_bam_files/Endo_P7_EctOvary_bulk.markdup.sorted.split.bam

```
 Repeat for all samples. 

 Rerun EHdn analysis.
